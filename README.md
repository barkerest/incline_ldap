# InclineLdap

The incline_ldap gem adds LDAP authentication to an [Incline](https://github.com/barkerest/incline) application.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'incline_ldap'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install incline_ldap
